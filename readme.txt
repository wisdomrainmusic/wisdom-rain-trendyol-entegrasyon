Trendyol mağazanızı WooCommerce üzerinden yönetmek için temel entegrasyon eklentisi. API ayarları ve bağlantı testi içerir.

== Description ==

Bu eklenti, Trendyol mağazanızı WooCommerce panelinden yönetmek için temel bir entegrasyon sağlar. İlk sürümde API kimlik bilgileri için ayar ekranı, Trendyol API istemcisi ve bağlantı testi bulunur.

> Not: Eklentinin çalışması için WooCommerce'in etkin olması gerekir.

== Installation ==

1. Eklentiyi indirip `wp-content/plugins/wisdom-rain-trendyol-entegrasyon` klasörüne yükleyin.
2. WordPress yönetici panelinde Eklentiler > Yüklü Eklentiler bölümünden eklentiyi etkinleştirin.
3. WooCommerce > Trendyol Entegrasyon menüsünden API kimlik bilgilerinizi girin.
4. Ayarları kaydedin ve "Test Connection" butonu ile bağlantıyı doğrulayın.
